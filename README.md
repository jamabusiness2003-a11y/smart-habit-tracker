# Smart Habit Tracker

## Description 
A web app that can track people's daily habit. It will help users build consistency and stay organized through a fast, intuitive interface.

The app focuses on performance and usability featuring a smart search system that allows users to quickly find and manage habits with minimal friction similar to the responsiveness of Google Search.

## App Features

### Core Features
- Add a habit
- Delete a habit
- Mark habit as complete
- View list of habit 
- Persistant data
- Track per day
